h1. Libres

Libre é um sistema para gerenciamento de livros idealizado para fazer uma prova de conceito do uso de Behavior Driven Development usando Java, Rails, PHP e .NET. Cada plataforma terá um fork nesse repositório com o código-fonte produzido.

h2. Projetos

* *master*: Documentação do projeto
* *libres-rails*: Implementação em Rails 
* *libres-java*: Implementação em Java 
* *libres-php*: Implementação em PHP 
* *libres-dotnet*: Implementação em .NET
